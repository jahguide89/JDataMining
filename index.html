Welcome to JDataMining

JDataMining is a open source Java library for data classification task.

Features

  • You can do data set management
  • JDataMining contains tools for data preprocessing
    • Normalize
    • Standardize
  • You can apply data classification techniques
    • Knn
    • Naive Bayes
  • You can generate experiments with training/test sets
    • Cross validation
    • n-fold cross validation
  • JDataMining is open source library issued under the GNU General Public License
